ClawBox Install Troubleshooting

ClawBox doesn't appear on my network after powering on

Wait 60 seconds for the full boot sequence. If using Wi-Fi, make sure you connected to the ClawBox hotspot first (it appears as "ClawBox-XXXX" during initial setup). If using Ethernet, try a different port on your router. The status LED should be solid green when the device is ready.

QR code won't scan or pairing fails

Make sure your phone and ClawBox are on the same Wi-Fi network. Try opening the pairing URL manually in your browser — it's shown below the QR code on screen. If you don't have a monitor connected, navigate to http://clawbox.local or check your router for the device's IP address.

How do I update ClawBox after the initial install?

ClawBox checks for updates automatically and applies them in the background. You can also trigger a manual update from the admin panel. Updates are delivered over-the-air and never require a reinstall — your conversations, settings, and custom skills are preserved.

Can I move ClawBox to a different network?

Yes. Hold the reset button for 3 seconds to enter network setup mode. ClawBox will broadcast its setup hotspot again, letting you connect to a new Wi-Fi network. Ethernet connections work immediately on any network.
